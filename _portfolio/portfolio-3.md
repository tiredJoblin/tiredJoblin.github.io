This is a rougelike mod for the game Chippy that I updated with new content monthy.

Tools I used: Visual Studio Code, C#, Steam Workshop, and Steam.

Skills I used: Game Design, Creativity, Coding Knowledge, Creative Problem Solving.

What I learnt: How to set up a system that can spawn certain enemies in a specific order and when certain conditions have been met, how to design enemies that can be fought in certain orders while still providing interesting challenges, and making sure not to overscope with an idea.

Challenges I solved: Figuring out how to insure enemies spawn only when the previous enemy has died, countless different bullet patterns, and making a system that can be explained with more enemies over time.
